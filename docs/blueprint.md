# **App Name**: Bitemates Landing Page

## Core Features:

- Hero Section: A clean hero section with a bold headline and a floating 3D iPhone mockup displaying the app's interface. Includes a prominent 'Get Started' CTA.
- Features Grid: A grid showcasing app features with white cards, subtle shadows, and minimalist line icons illustrating key functionalities.

## Style Guidelines:

- Background: Clean White (#FFFFFF) for a spacious and breathable feel.
- Primary Accent: Vibrant Iris Blue (#4E46DC) for download buttons, gradient overlays, and key graphical elements.
- Text: Dark Gray (#333333) for headings and body text, ensuring high readability.
- Headline font: 'Poppins' for a contemporary, precise feel. This font is a sans-serif.
- Body font: 'Inter' for a modern, machined, objective, neutral feel. This font is a sans-serif.
- Employ significant negative space to enhance clarity and focus.
- Use soft rounded corners (24px radius) throughout the design to create a friendly, approachable aesthetic.
- Incorporate subtle glassmorphism effects on app mockups for a modern, layered appearance.
